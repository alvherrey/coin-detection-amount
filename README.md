## Requerimientos
### Crear un entorno virtual con Miniconda
Creamos un entorno nuevo con python 3.8
```bash
conda create --name coin-detecion-amount python=3.8
```
Activamos el entorno
```bash
conda activate coin-detecion-amount
```
